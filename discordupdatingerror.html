Discord updating error

Sometimes Discord will fail to update if third party client mods are installed. Here's how to fix it when using Vencord. Keep in mind that this is different than the other Discord updating bug: https://www.youtube.com/watch?v=3OEfr7L-gUk

1. Backup your Vencord folder

To find your Vencord folder, open Settings in Discord, go to the "Vencord" tab, and click "Open Settings Folder". Then, copy it to a safe directory (eg: Downloads)

2. Uninstall Vencord

To run the Vencord uninstaller, run the launcher script from https://vencord.dev/download/, then click "Uninstall" and "Uninstall OpenAsar".

3. Update Discord

Fully close Discord via the system tray icon and update your system using your system updater of choice.

4. Reinstall Vencord

Run the Vencord installer script from https://vencord.dev/download/ and install Vencord (I highly recommend installing OpenAsar too)

5. Import configs

Open Settings in Discord, go to the "Vencord" tab, and click "Open Settings Folder". Then, find the Vencord folder that you backed up and copy the contents of it to the active Vencord folder, then restart Discord. Your Discord should now have the same plugins and themes as before.
